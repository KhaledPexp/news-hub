Var is the oldest ways to declare a variable in javascript and it is accessible from both global and functional scope. Also We can reassign and redeclare var. Let and Const variable are accessible only in block scope. Let variable accept reassigninng a value but const doesn't allow it. Also We can't redeclare let and const.

Arrow function is the newest way of function declation in javascript. syntax of regular and arrow function are different. Arrow function doesn't have their own this keyword like regular function. Arguments object are not available in arrow function but available in regular funciton. Regular functions can be used as constructors, using new keyword but arrow funciton can't.

Map takes a callback and run against every element on array and generate a new array based on the existing array. Filter take a callback for every element and returns a filtered array.Filter executes the callback and check its return value and it the value is true element remains in the resulting array but if it false then it's remove from resulting array. find and filter work like same but find returns the first matching element and filter returns an array of all matching elements foreach takes a callback function and run on each element of array one by one.foreach works like tradional for looping.

Template literals are declared with backtick (``) characters. Template literals make it far easier to write multiline strings then normal string declartion. It's easier to include varialbes and expression inside a string using template literals. syntax like ${..}.A more advanced form of template literals are tagged templates. Tags allow you to parse template literals with a function.
